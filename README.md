# Answers to 5 Questions

### 1. What is the difference between var, let, and const?

**Answer:**  
`var` is function-scoped and was used in older JavaScript.  
`let` is block-scoped and allows updating the value.  
`const` is also block-scoped but its value cannot be reassigned after initialization.

---

### 2. What is the spread operator (...)?

**Answer:**  
The spread operator (`...`) is used to expand elements of arrays, objects, or strings.  
It helps to copy, merge, or pass multiple values easily.

---

### 3. What is the difference between map(), filter(), and forEach()?

**Answer:**  
`map()` creates a new array by transforming elements.  
`filter()` creates a new array with elements that match a condition.  
`forEach()` runs a function for each element but does not return a new array.

---

### 4. What is an arrow function?

**Answer:**  
An arrow function is a shorter syntax for writing functions in JavaScript using the `=>` symbol.  
It makes code cleaner and more concise.

---

### 5. What are template literals?

**Answer:**  
Template literals allow easier string creation using backticks (`` ` ``).  
They support multi-line strings and variable interpolation using `${}`.
